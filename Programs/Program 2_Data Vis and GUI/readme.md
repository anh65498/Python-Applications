## Getting Started

#### Install modules
For Python3, install these modules in CLT first
```
$ pip3 install numpy
$ pip3 install matplotlib
```

If you see no graph:
```
$ touch ~/.matplotlib/matplotlibrc
Open file matplotlibrc and add
  backend: TkAgg
into the file, then save and quit
```
